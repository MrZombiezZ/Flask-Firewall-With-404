## Getting Started
You will need to have Flask installed for the application and modules.
```
> mkdir myproject
> cd myproject
> py -3 -m venv venv
```
To activate the corresponding environment :
```
> venv\Scripts\activate
```
To install Flask after the commands above :
```
$ pip install Flask
```
